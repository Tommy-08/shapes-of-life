Beyond the Print: A Kinetic Homage

While a standard print is a static reproduction, a plot is a physical performance. Each hash mark represents a distinct moment in an iconic life; by drawing them separately, the plotter creates a foundational, physical record of time itself.

The 8-hour duration isn't just a technical requirement—it's a deliberate act of patience and a rhythmic tribute to the subject's enduring legacy.

Phase 01

The Algorithm

Every hash mark is rooted in data. We ingest quantitative records of life events—durations, intensities, and timestamps—converting them into variables for procedural generation.

A custom Python script processes these inputs using Perlin noise and mathematical constants to derive the final visual path.

path_generator.py
# Hash Generation Logic
def generate_path(seed, complexity):
    points = []
    for i in range(complexity):
        x = math.sin(seed * i) * (i * 0.1)
        y = math.cos(seed * i) * (i * 0.1)
        points.append((x, y))
    return points

Phase 02

The Layout

The transition from code to canvas requires human intuition. We export the generated vector paths to Adobe Illustrator for architectural alignment.

Every 19x24 inch canvas is hand-tuned, adjusting line weights and spatial relationships to ensure the composition breathes before the mechanical plot begins.

Phase 03

The Plot

Execution is a test of endurance. We utilize the AxiDraw V3/A3 pen plotter for its unmatched mechanical precision and rhythmic consistency.

A single archival record requires approximately 8 hours of continuous plotting. During this time, the machine captures the subtle vibrations of the studio, embedding a unique signature into every line.

Phase 04

The Materials

Permanence is our priority. We utilize 300gsm acid-free cotton rag paper, chosen for its structural integrity and tactile grain.

Our drawings are rendered with lightfast carbon-based pigment ink (ISO 12757-2). This combination ensures that the record remains as vivid a century from now as the day it was plotted.
