The Original Code.

A multi-year journey of writing and discovering exactly what I wanted from generative systems.

Writing the original algorithm that powers Shapes of Life was a project that spanned almost two years (2024-2026). It started as a simple Python script looking at basic vector outputs, but quickly evolved into an obsession with mathematically capturing history.

"The goal wasn't just to make data look pretty. The goal was to write logic deterministic enough to map an entire life faithfully, and chaotic enough to simulate gravity."

Initially, I experimented with raw coordinates, just trying to feed JSON life events into an array that a plotter could comprehend. But lines quickly look mechanical if they lack rhythm. I spent the entirety of 2025 tweaking exactly how the "Hash Marks" felt on the page. How does it look when an artist takes a three-year hiatus? How does the code draw a line differently if a milestone is particularly monumental?
